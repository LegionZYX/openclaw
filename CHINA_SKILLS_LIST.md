## 🚀 安装方法

### 方式 1: 通过 ClawHub 安装（推荐）

```bash
# 安装单个技能
openclaw skill install <skill-name>

# 示例
openclaw skill install content-creator
openclaw skill install local-whisper
```

### 方式 2: 手动下载

```bash
# 1. 访问 ClawHub 链接
# 2. 下载 SKILL.md 文件
# 3. 放到 ~/.openclaw/workspace/skills/<skill-name>/
```

### 方式 3: 使用 clawhub 命令

```bash
# 列出所有可用技能
openclaw clawhub list

# 搜索技能
openclaw clawhub search "关键词"

# 安装技能
openclaw clawhub install <skill-slug>
```

## 📖 配置示例

### 飞书集成

```json5
// ~/.openclaw/openclaw.json
{
  skills: {
    entries: {
      "feishu-calendar": {
        enabled: true,
        env: {
          FEISHU_APP_ID: "cli_xxxxxxxxxxxx",
          FEISHU_APP_SECRET: "xxxxxxxxxxxxxxxx",
        },
      },
    },
  },
}
```

### 本地语音识别

```json5
{
  skills: {
    entries: {
      "local-whisper": {
        enabled: true,
        config: {
          model: "base",
          language: "auto",
        },
      },
    },
  },
}
```
